## 🚀 Featured Projects

### 🔏 [SealSight — AI Image Watermarking](https://github.com/abir1807/SealSight)
> Semantic-aware, crop-resistant watermarking for AI-generated images.

- Embeds an **invisible 32-bit signature** into the core subject of any image using **DCT frequency-domain encoding**
- Verifies watermarks even after the image is **cropped or compressed**
- Subject-first approach: watermark targets the **semantically important region**, not scattered across the whole image
- Stack: `Python` `FastAPI` `HTML` `Dockerfile`

---

### 🛎️ [QuickServe — Local Service Booking Platform](https://github.com/abir1807/QuickServe)
> A full-stack service marketplace connecting users with verified local service providers.

- Users can **discover and book** local services across multiple categories
- Service providers get a **dedicated dashboard** to manage listings and bookings
- Admin panel for **full control** over providers, services, categories, and bookings
- Stack: `Spring Boot` `React` `MySQL` `Java` `CSS`

---

### 🤟 [Sign Language Detector](https://github.com/abir1807/sign-language-detector)
> Real-time ASL gesture recognition using MediaPipe + TensorFlow.

- Detects **American Sign Language (ASL)** gestures in real time via webcam
- Built with **MediaPipe** for hand landmark detection and **TensorFlow** for classification
- Structured with `app/`, `data/`, `models/`, `src/` for clean project architecture
- Stack: `Python` `TensorFlow` `MediaPipe`

---

### 💊 [PharmaFast — Online Medicine Management System](https://github.com/abir1807/Pharmafast-online_medic_system)
> Online Medicine Management System built with Advanced Java following MVC architecture.

- Digitizes **medicine ordering, appointment booking**, and video consultations
- Includes **location-based emergency assistance** for healthcare users
- Built with **Java Servlets + JSP** following the MVC pattern
- Stack: `Java` `JSP` `Servlets` `MVC` `MySQL`
